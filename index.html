What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure of web pages using a variety of tags and attributes. HTML elements are the building blocks of HTML pages and can include text, images, links, and other content. HTML is essential for creating and structuring content on the web.

A basic HTML document includes the following structure:

Let us now look at the structure :

<!DOCTYPE html>
<html>
<head>
<title>My First Web Page</title>
</head>
<body>
<h1>Hello World!</h1>
<p>Welcome to my first web page.</p>
</body>
</html>

HTML provides the foundation for web development and is often used in conjunction with CSS and JavaScript to create interactive and visually appealing web pages.

CSS Basics

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML. CSS controls the layout, color, fonts, and overall appearance of web pages. It allows developers to separate the content of a web page from its visual presentation.

CSS Syntax: CSS consists of selectors and declaration blocks. A selector is used to target HTML elements, and the declaration block contains one or more property-value pairs.
Example CSS:

h1 {
color: blue;
text-align: center;
}
p {
font-size: 16px;
line-height: 1.5;
}

CSS enhances the visual appeal of web pages and helps in creating responsive designs that adapt to various screen sizes.

CSS Selectors

CSS selectors are used to target HTML elements to apply styles. They can be simple or complex, depending on the needs of the design.

Common CSS Selectors:

Element Selector: Targets elements by their tag name. Example: p { color: red; }

Class Selector: Targets elements with a specific class attribute. Example: .class-name { font-size: 20px; }

ID Selector: Targets elements with a specific id attribute. Example: #id-name { background-color: yellow; }

Attribute Selector: Targets elements based on their attributes. Example: [type="text"] { border: 1px solid gray; }

Box Model

The CSS Box Model is a fundamental concept that describes how elements are structured and how their sizes are calculated. The box model consists of:

Content: The actual content of the box, such as text or images.

Padding: The space between the content and the border. Padding can be set on all sides of the box.

Border: A border surrounding the padding (if any). Borders can have different styles, widths, and colors.

Margin: The space outside the border, separating the element from other elements. Margins can be used to create space between elements.

Flexbox

Flexbox is a CSS layout module designed to help create flexible and responsive layouts. It allows items in a container to be aligned and distributed efficiently within the available space.

Key Flexbox Properties:

Container Properties:

  • display: flex; - Defines a flex container.
  • flex-direction - Specifies the direction of the flex items (row, column, row-reverse, column-reverse).
  • justify-content - Aligns items along the main axis (flex-start, center, space-between, etc.).
  • align-items - Aligns items along the cross axis (flex-start, center, stretch, etc.).
Item Properties:
  • flex-grow - Defines the ability of a flex item to grow relative to other items.
  • flex-shrink - Defines the ability of a flex item to shrink relative to other items.
  • flex-basis - Defines the default size of a flex item before growing or shrinking.

CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web. It allows developers to create complex layouts using rows and columns. Grid layout enables precise control over element placement and alignment.

Key Grid Properties:

Container Properties:

  • display: grid; - Defines a grid container.
  • grid-template-columns - Defines the number and size of columns.
  • grid-template-rows - Defines the number and size of rows.
  • grid-gap - Defines the gaps between grid items.
Item Properties:
  • grid-column - Defines how many columns an item should span.
  • grid-row - Defines how many rows an item should span.
  • justify-self - Aligns an item horizontally within its grid area.
  • align-self - Aligns an item vertically within its grid area.
